## 1. 機能概要

掲示板機能は **二系統** で理解する。第一に、ユーザーが愚痴・改善提案・論文議論などを行う **製品 BBS**（Kernel 動的ルート `/board/*` 等）。第二に、各 Markdown 正本に **1 ファイル = 1 掲示板スレ** を紐づけ、意図・検討・廃止・採用を残す **file-board linkage**（REQ-018）。後者は legacy civ-os 向け RAG 索引基盤であり、**IHL では GitHub（PR · BOARD · Discussions）+ R2** が正本（C-Sync は legacy のみ）。前者の UX 救済（投稿失敗・再試行・保存境界）は REQ-024 の **BBS posting rescue** スライスとして未完。コンポーネント掲示板（§19）とは名称が oral で重なるが、REQ-018 の正本は **MD↔boards/** である。

## 2. ユーザーができること

### 製品 BBS（ユーザー向け · H-BBS 確定 2026-06-07）

**主入口は 4 つのみ**（[ADR-H-07](../02-設計/_横断/adr/ADR-H-07-掲示板-入口4つ-論文内研究.md)）。**ユーザー向け第一入口**は **知の広場 · 掲示板タブ**（`/knowledge/board` → 下表 4 ルート）:

| 入口 | ルート（案） | 目的 |
|------|--------------|------|
| **愚痴** | `/board/complaint` | 感情整理 |
| **改善** | `/board/improvement` | 機能改善提案 |
| **論文** | `/board/paper` | 研究・論文 · **観察・繁殖ログ等の長文もここ**（独立 Research Board **なし**） |
| **その他** | `/board/general` | 上記外 |

- **論文板内ケース分け**: 投稿時に `paper_case`（enum）または tag で **observation · breeding_log · analysis · review · replication · hypothesis** 等を選択。一覧は **case チップ**でフィルタ（FR-BBS-15/16）。
- **Research Board（独立板）**: **v1 不採用** — 量が足りない場合のみ将来 ADR で再検討（likely never）。
- **マーケット Engagement 争い**（Q&A・称賛・ラブレター等）: **掲示板と同型の指摘**（[`11-裁判.md`](./11-裁判.md) §6.5 · U-MKT-DSP-09 解決）— 特別フローなし。
- **裁判・スレッド**（`/board/justice`）: 司法関連スレッド（menuConfig 表記）。
- **説明掲示板**（`/manual` hybrid）: 利用者向けマニュアル系。

### file-board linkage（開発・RAG 向け）

- 正本 MD ごとに `boards/P/p15_file_linkage/{dev|manual}/*.md` スレを参照・追記。
- Twin RAG で **file_board** / **file_board_thread** 種別検索。
- `generate-file-board-linkage.ts --apply` で新規正本にスレ追加・CSV 同期。

## 6. MiniKernel / C-USB 上の位置づけ

- **FeatureNode**: `board`（Governance/R2 レイヤと併記）。
- **Kernel**: 動的掲示板は **MiniScreenKernel** 経由（`ROUTING_TABLE`・registry renderer）。画面概念は Kernel UUID ルーティング。
- **Component**: 各掲示板種別は fork 可能な Component。file-board スレ自体は **P 層 Process**（`p15_file_linkage`）の成果物。
- **ITO（file-board）**: IN = 正本 MD パス → Transform = テンプレ生成・SHA256 短縮ファイル名 → OUT = boards/*.md + CSV 行。
- **legacy C-Sync（civ-os のみ）**: 掲示板は 4 媒体同期の **post** 媒体 — **IHL 不採用**（GitHub + R2 に置換）。
