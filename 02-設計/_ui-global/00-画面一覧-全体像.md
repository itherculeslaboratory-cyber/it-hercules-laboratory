## 1. 共通デザインシステム（血統 OS · `ui-reference/preferences.md` §A/§B/§C）

| 項目 | 値 |
|------|-----|
| 背景（深） | `#0D0D0D` |
| セクション | `#121212` |
| カード | `#1A1A1A` · 枠 `1px #2A2A2A` · 角丸 12px · **影なし** |
| 本文 / 補足 / 無効 | `#E6E6E6` / `#B3B3B3` / `#666666` |
