## 配置手順

1. [`00-世界観アセット一覧-v1.md`](../00-世界観アセット一覧-v1.md) で `id` とパスを確認  
2. 該当サブフォルダにファイルを配置（ファイル名は一覧の「配置パス」に合わせる）  
3. 一覧の `status` を `user_placed` に更新  
4. 関連 UI 設計 md に相対パスリンクを 1 行追加
