# ADR-H-07: 掲示板 — 入口 4 つ · 論文内研究（H-BBS）

> **ステータス**: 採用（人間確定）  
> **決定日**: 2026-06-07

## 文脈

2026.06.08 アイディア §3–4 は **Research Board と Community Board の分離**を示唆していた。ユーザー確定（2026-06-07）では **製品 BBS はシンプル・直感的** — 入口 4 つに集約し、研究・観察の長文は **論文板の中**でケース分けする。

> ユーザー引用: 「研究は論文内でしょ？ どう考えても」

---

## 決定

### 1. 製品 BBS 入口 — **4 つだけ**

| 入口 | ルート（案） | 目的 |
|------|--------------|------|
| **愚痴** | `/board/complaint` | 感情整理 |
| **改善** | `/board/improvement` | 機能改善提案 |
| **論文** | `/board/paper` | 研究・論文・観察記録の **唯一の長文入口** |
| **その他** | `/board/general` | 上記外 |

- **Research Board（独立板）は作らない**
- **裁判** · **説明** は従来どおり **二次導線**（FR-BBS-06 併設 · 主 4 板に含めない）

### 2. 論文板内 — ケース分け（tags または subcategory enum）

観察ログ · 繁殖記録 · 分析 · 査読 · 追試などは **論文タブ内**でフィルタする。

| case（enum 案） | 用途 |
|-----------------|------|
| `paper` | 論文・preprint 共有 |
| `observation` | 観察記録・フィールドノート |
| `breeding_log` | 繁殖・交配ログ |
| `analysis` | データ分析・統計 |
| `review` | 査読・コメント |
| `replication` | 追試・再現報告 |
| `hypothesis` | 仮説提示 |
| `other` | 上記外（論文板内） |

- UI: **case チップ**（フィルタ）+ 投稿時 **case 選択（必須）**
- データ: `board_category=paper` + `paper_case`（または tag 配列 · 詳細設計で単一 enum を推奨）
- [`09-論文.md`](./09-論文.md) の Content 型（paper, review, hypothesis 等）とは **別層** — BBS スレッド分類。相互リンクは詳細設計

### 3. Research Board — 延期

| 項目 | 内容 |
|------|------|
| **独立 Research Board** | **v1 スコープ外** |
| **再検討条件** | 論文板 + case チップで **量が足りない** と判明したときのみ |
| **見込み** | 論文 + tags で足りる想定（**likely never**） |

2026.06.08 アイディア §3 の「Research Board と Community Board を分離」は **IHL 製品 BBS では採用しない**（研究 Content パイプラインは 09 · ADR-H-04 で別途 Truth 化）。

## UX 原則（確定）

- ハブ **4 タブ**で入口を固定（[`07-掲示板-遷移詳細.md`](../../02-設計/features/07-掲示板/ui/遷移詳細.md)）
- 論文板のみ **case チップ行**を追加（他 3 板には置かない）
- 投稿先取り違え防止: パンくず + 左ナビハイライト（既存 FR）

## 影響

- **UI 設計** — 論文モックに case チップを追記（PNG 再生成は別タスク）
- **11 裁判** 指摘入口 — 変更なし（全板同型）
