# ADR-H-14 — グローバル文脈バー（掲示板 · テンプレ · Builder）

> **ステータス**: 草案 v1（2026-06-09 · UX レビュー Step 24 反映）
> **背景**: ユーザー指摘 — 各画面から「この画面向け」の愚痴板・改善提案板・テンプレ一覧へ飛べず、使いづらさをその場で書けない。

---

## 決定

全主要画面の **ヘッダー右（またはフッター固定）** に **文脈バー（Context Bar）** を常設する。内容は **現在の route / screen_id** で自動絞り込み。

| 入口 | 遷移先 | クエリ例 |
|------|--------|----------|
| 愚痴を書く | `/board/gripe` | `?context_route=/observation` |
| 改善を提案 | `/board/:category`（改善提案） | `?context_route=/observation` |
| この画面のテンプレ | `/observation/templates` または Builder テンプレ索引 | `?scope_route=/observation` |
| UI を編集 | `/builder` | `?edit_route=/observation` |
| テンプレを適用 | 同一画面（モーダル） | `template_id` + `apply_to_route` |
| **種を選ぶ / 切替**（観測ドメインのみ） | 05ctx ボトムシート | `species` + `stage` + `scope_route`（[ADR-H-15](./ADR-H-15-観測コンテキスト.md)） |

## 観測拡張（ADR-H-15）

観測ドメイン（`/observation*`）では文脈バーに **種族チップ** を追加し、**観測コンテキスト（WorkflowContext = species + stage_name + 任意 phase）** を 1 度決めて 05i/05a/05tl に引き継ぐ。

- 種族チップ 例: 〔🪲 ヘラクレス · L3 幼虫〕→ タップで **ボトムシート 05ctx**（全画面遷移にしない）。未設定時は〔＋ 種を選ぶ〕。
- チップは **観測ドメインのみ**表示（他ドメインの文脈バーには出さない）。**副次・等サイズ**で 1 主ボタン原則を侵さない。
- 詳細契約は [ADR-H-15](./ADR-H-15-観測コンテキスト.md)、ピッカー UI は [`../../02-設計/features/05-観測/ui/コンテキスト.md`](../../02-設計/features/05-観測/ui/コンテキスト.md)。

## 原則

- **3 クリック以内**: 文脈バーは 1 クリックで板 or テンプレ一覧へ。
- **1 画面 1 主ボタン**は各画面本体のまま。文脈バーは **副次・等サイズのアイコン+短文**（4 入口まで）。
- テンプレ一覧は **遷移元 URL でフィルタ**（全画面テンプレは別導線）。
- Fork → 登録 → 適用は ADR-H-13 · ADR-H-01 に従う。
