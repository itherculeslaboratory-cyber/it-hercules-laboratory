## 1. 主タスク（1 つ）

**条項を読んで同意の意思を確認する**（閲覧のみルートでも同 UI）。

---

## 2. イメージ画面

![利用規約](./mockups/ihl-00-terms.png)

| チャンク | 内容 |
|----------|------|
| **ヘッダ** | パンくず「オンボーディング › 利用規約」· タイトル |
| **条項本文** | スクロール可能カード 3〜5 節（プレースホルダ可 · 正本は `docs/legal/`） |
| **条項ごと解説** | 各条見出し横に **〔本音解説〕**（小学5年生向け短文）+ **解説動画**リンク（青 `#4DA3FF`） |
| **兄弟リンク** | プライバシー · 市場注意書（青 `#4DA3FF`） |
| **フッタ** | 副「戻る」· **主「同意して続ける」**（緑 · 登録画面から来た場合のみ有効） |

---

## 3. オンボーディングとの関係

- O2 登録画面のチェック行から **同一 mock** へ遷移（[`ux-walkthrough/`](./ux-walkthrough/) 参照）
- 同意は **O2 で API 送信** · O3 は閲覧+確認 UI

---

## 4. 状態（必須）

| 状態 | 表示 |
|------|------|
| loading | 条項スケルトン |
| error | 再試行 + 登録へ戻る |

---

## 5. デザイン規範

- 背景 `#0D0D0D` · カード `#1A1A1A` · [`ui-reference/preferences.md`](../../../ui-reference/preferences.md) §B
- 法務文言は **装飾しない**（意味色のみ · リンク青）
