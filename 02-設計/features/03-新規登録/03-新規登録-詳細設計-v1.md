## 1. API 契約

| メソッド | ルート | 内容 |
|---------|--------|------|
| `POST` | `/api/auth/onboarding` | handle · display_name · locale · agree_terms |
| `GET` | `/api/auth/handle-available` | handle 重複チェック |

### 1.1 handle 規則

- 3〜32 文字 · `[a-z0-9_.]` · 先頭英字
- 予約語リスト（admin · system 等）

### 1.2 locale

- 登録時確定 · 以降は 12 設定 + 21 FR で変更可
