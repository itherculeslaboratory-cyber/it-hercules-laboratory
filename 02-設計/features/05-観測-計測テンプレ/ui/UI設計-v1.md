## 2. 05tl — テンプレ一覧 LIST

### 2.1 レイアウト（3〜4 チャンク）

1. **ヘッダ**: パンくず「観測 › 計測テンプレ」· 1 主ボタン **〔新規テンプレを作成〕**
2. **コンテキストバナー（OBS-CTX / OBS-TGT · 条件表示）**: WorkflowContext の **対象（`ObservationTarget`）** がある時のみ「**○○向けテンプレ**」等の対象バナー + **絞り込み件数**（例: 〔ヘラクレス（原名亜種）· L3 幼虫〕に `target_scope` 合致 **8 件**）+ 〔絞り込み解除〕。詳細は §2.5。**文字のみ**（絵なし）。
3. **フィルタ行**: タブ〔すべて | 自分 | 公開 | フォーク済〕· **ドメイン/対象スコープ チップ**（生物・器物・デジタル…/ `target_scope` 接頭辞）· 性別チップ〔雄 | 雌 | 共通〕· 検索（名前）

> **target_scope と性別の分離**: テンプレ絞り込みは **`target_scope`（domain + 分類パス接頭辞）** が主（OBS-TGT-07）。**性別チップ**は表示分岐の補助。**令（L1〜L3）・発育段階はフィルタに置かず入力 05i で確定**（ADR-H-16 §6）。
4. **カードグリッド**: テンプレカード（名前 · 説明 1 行 · 項目数 · 更新日 · 可視性バッジ · フォーク元リンク）

### 2.2 カード 1 枚の情報

| 要素 | 内容 |
|------|------|
| タイトル | テンプレ名（例: 雄 標準計測 v2） |
| 説明 | 最大 1 行（truncate） |
| メタ | 項目 **N** 行 · 最終更新 · 作成者 handle |
| バッジ | `公開` / `自分のみ` / `フォーク`（parent 名リンク） |
| 副次 | 〔詳細を見る →〕（カード全体タップ = 詳細へ） |

### 2.5 観測コンテキスト・対象スコープのプリフィル（OBS-CTX / OBS-TGT · [ADR-H-15](./ADR-H-15-観測コンテキスト.md) · [ADR-H-16](./ADR-H-16-観測対象ナビゲータ.md)）

- 受け口: `/observation/templates?target_id=<ULID>&domain=<...>&stage=<egg|larva|pupa|adult>&scope_route=/observation`（旧 `?species=` は後方互換）。
- コンテキストの **対象**があると、`target_scope`（domain + 分類パス接頭辞）が合致するテンプレを **上位表示**し、対象バナー（文字のみ）と合致件数を表示する（OBS-TGT-07）。
- マッチ規則: `ObservationTarget.domain == target_scope.domain` かつ `path` が `target_scope.path_prefix` で始まる（ADR-H-16 §6）。`path_prefix=[]` は「ドメイン全体」。
- バナーの 〔絞り込み解除〕 は **この一覧のフィルタのみ解除**（WorkflowContext 自体は消さない — 他画面に影響しない）。
- コンテキスト無し（URL/localStorage/profile すべて空）時は **バナー非表示**で従来どおり全件 + フィルタ。
- 優先順は **URL クエリ > localStorage > profile**（ADR-H-15 §5）。

## 3. 05td — テンプレ詳細 DETAIL

### 3.1 レイアウト（3 チャンク）

1. **ヘッダ**: パンくず「観測 › 計測テンプレ › {名前}」· 可視性バッジ
2. **メタブロック**: 説明文 · 作成者 · 更新日 · フォーク元（あれば）· 使用回数（任意）· **対象スコープ（`target_scope`: ドメイン + 分類パス接頭辞 · 文字表示）**（OBS-TGT-07）
3. **項目リスト（読取）**: 行 = 項目名 · 既定単位 · 既定計測方法 · 性別適用（both/male/female）
4. **アクション行**（副次は小ボタン · **1 主ボタン**）:
   - **主**: 〔このテンプレで記録〕→ `/observation/input?templateId=`
   - 副: 〔複製して編集（Fork）〕· 〔編集〕（自分のみ）· 〔削除不可 — 非公開化のみ〕

### 3.2 Fork との関係

- **詳細画面から Fork** = 〔複製して編集〕→ 子テンプレ編集モード（既存 `ihl-05-obs-template-fork.png` の編集ペインと同型）→ 保存で `TemplateForkEvent`。
- **一覧からは詳細経由**（一覧に Fork 直ボタンは置かない — 誤 Fork 防止）。

### 3.3 状態

| 状態 | 表示 |
|------|------|
| loading | メタ + 行スケルトン |
| empty | 404「テンプレが見つかりません」→ 一覧へ |
| error | 再試行 |
| 権限なし | 公開テンプレは閲覧可 · 編集/Fork 不可（Fork は可 — OBS-TPL-11） |
