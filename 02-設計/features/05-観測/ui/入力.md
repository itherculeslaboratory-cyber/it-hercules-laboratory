## 1. 主タスク（1 つ）

**「項目・数値・単位・計測方法を、性別に合ったテンプレで素早く記録する」**（1 主ボタン〔保存〕・3 クリック以内）。

---

## 2. イメージ画面

### 2.1 計測入力行（4 コントロール）

![計測入力行（4 コントロール）](./mockups/ihl-05-obs-input-row.png)

| コントロール | 仕様 |
|--------------|------|
| **項目** | ドロップダウン（候補）＋「＋ 自由入力」（`measurement_name.yaml`） |
| **数値** | 空入力（`value_type` で種別変化） |
| **単位** | ドロップダウン（候補）＋「＋ 追加」（`unit_default` 初期） |
| **計測方法** | ドロップダウン（手入力 / IoT取得 …）（`measurement_method.yaml`） |

### 2.2 雄テンプレ（角長 表示）

![計測入力 雄（角長 表示）](./mockups/ihl-05-obs-input-male.png)

性別トグル〔雄〕active。`sex_visibility_rule.male` で **体重・体長・角長・胸幅** を既定表示。

### 2.3 雌テンプレ（角長 非表示）

![計測入力 雌（角長 非表示）](./mockups/ihl-05-obs-input-female.png)

性別トグル〔雌〕active。**角長（male 専用）を既定で淡色/非表示**（「雌では既定で非表示」注記）。値は破棄しない。

### 2.4 テンプレ Fork（選択 → 複製 → 編集）

![テンプレ Fork フロー](./mockups/ihl-05-obs-template-fork.png)

一覧 → 〔このテンプレを複製して編集〕 → 〔保存〕の **3 クリック**。保存時 `TemplateForkEvent` INSERT（ADR-H-04 §7）。

### 2.5 IoT 選択 × 機器未登録（機器管理へ）

![IoT 選択・機器未登録バナー](./mockups/ihl-05-obs-device-link.png)

計測方法 IoT かつ機器未登録時のみ、注意色バナー「機器が未登録です」+ 大ボタン〔機器管理へ〕（`device_route`）。

---

## 3. 状態（必須）

| 状態 | 表示 |
|------|------|
| loading | 行スケルトン・テンプレ一覧スピナ |
| empty | テンプレ未選択「テンプレを選ぶ or 空で項目追加」/ 行 0 件は〔行を追加〕強調 |
| error | 「保存できませんでした」+ 再試行（raw 非表示）・該当行ハイライト |
| 権限なし | 他人テンプレは閲覧のみ・複製は可 |

---

## 4. デザイン規範（§A/§B/§C）

- 背景 `#0D0D0D` · カード `#1A1A1A`/枠 `#2A2A2A`/角丸 12px/影なし · ボタン角丸 8px。
- **1 画面 1 主ボタン**（保存）· 大ボタン · 3 クリック以内。
- **色は意味のみ**（注意 `#FFD66B` · 成功 `#5CD68D` · 情報 `#4DA3FF`）。
- `value_origin` 混同禁止（計測方法 → value_origin 既定写像・保存時確定）。
