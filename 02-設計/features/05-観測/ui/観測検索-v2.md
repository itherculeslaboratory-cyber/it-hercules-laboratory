## 1. 共通シェル（filter → grid → detail）

```text
/observation
  ├─ フィルタ Card（species · sex · 絞り込む）
  ├─ 結果グリッド（capture カード · 4:3 プレースホルダ or blob サムネ）
  └─ 各カード → /observation/:capture_id

/observation/:capture_id
  ├─ ヘッダ（表示名 · capture_id · 観測日時 · 個体リンク）
  ├─ 動的セクション（存在ベース · 下記 §2）
  └─ 類似個体 · reanalysis-manifest 導線
```

**data-testid（Tier B）**: `obs-grid-page` · `obs-open-detail` · `obs-detail-page` · `obs-detail-measurements` · `obs-detail-photo` · `obs-detail-photo-conditions` · `obs-detail-env-snapshot` · `obs-detail-devices` · `obs-detail-similar` · `obs-detail-manifest-link`

## 2. 動的セクション registry（存在ベース描画）

| セクション ID | 表示条件 | データ源 | ver2 |
|---------------|----------|----------|------|
| `photo` | `image_url` または blob 存在 | capture `image_path` + GET `…/image` | **IN** |
| `measurements` | `measurements.length > 0` | Truth `capture/measurement` 縦持ち | **IN**（1〜100 行 · 折りたたみ可） |
| `photo_conditions` | `photo_conditions.length > 0` | `capture/photo_condition` | **IN** |
| `environment_snapshot` | snapshot オブジェクトあり（B モデル: 写真なし commit 時） | capture + `capture/environment_snapshot` | **IN**（表のみ） |
| `devices` | `devices.length > 0` | `observation_context.devices[]` | **IN** |
| `similar` | locator 有時 hits | `search_similar` | **IN**（本番 rerank は ver5） |
| `manifest` | capture 存在 | GET `…/reanalysis-manifest` | **IN**（最小メタ） |
| `timeseries_graph` | — | `environment_timeseries` | **OUT ver4+**（説明リンクのみ可） |

**20〜100 行 UX**: 10 行超で `<details>` 折りたたみ（先頭 10 行表示 + 「残り N 行」）。`group` / `device_id` によるグルーピングは ver3 任意。

## 3. ウィジェット境界（snapshot vs 時系列）

| ウィジェット | Phase | 内容 |
|--------------|-------|------|
| **環境 snapshot 表** | ver2 IN | `temperature_c` · `humidity_pct` · `source` · `captured_at` 等を 1 表 |
| **撮影条件表** | ver2 IN | `photo_conditions[]` 行リスト（色補正なし注記） |
| **時系列グラフ** | ver4+ OUT | プレースホルダ: 「個体の環境推移は今後のバージョンで表示」+ `/individuals/:id` 導線 |

## 4. Streamlit 5 pane → Next.js 差分

| Streamlit pane | Next.js ver2 |
|----------------|--------------|
| 検索フィルタ | `/observation` フィルタ Card |
| 結果グリッド | 同一ページ grid |
| 詳細メタ | `/observation/[id]` ヘッダ |
| 計測表 | 動的 `measurements` セクション |
| 類似 | `similar` セクション |

Streamlit UI は ver2 DONE 条件 **OUT**（Next.js のみ）。
