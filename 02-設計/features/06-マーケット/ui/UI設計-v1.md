## 3. 要点

- 三チャネルタブ（出品/オークション/テンプレ · FR-MKT-01）。状態チップは意味色のみ。
- プライベートボードは **当事者2人のみ・第三者非公開**（掲示板二人部屋とは別物 · §11.0 Stage 1）。
- ステッパ: マッチング→振込→配送→評価。支払期限 **2 週間**（Y01/Y02）。
- **8% 振込案内は Stage 3（取引成立）後**（§11.0.1 · [`23-GMO銀行振込判定.md`](./23-GMO銀行振込判定.md)）。
- PII（住所・口座）はボードに直書きせず設定で事前登録（FR-DSP-19）。
- 状態: loading / empty / error / 未ログイン 必須。
