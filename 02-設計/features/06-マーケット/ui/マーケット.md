## 2. イメージ画面

### 2.1 出品一覧（listing browse）

![マーケット 出品一覧](./mockups/ihl-06-market-browse.png)

| チャンク | 内容 |
|----------|------|
| **チャネルタブ** | 出品 / オークション / テンプレ（三柱 · FR-MKT-01） |
| **フィルタ** | 種 / 価格帯 / 並び替え |
| **出品カード** | 写真 / タイトル / 価格 / 状態チップ（出品中=緑・オークション=黄・成約=グレー）/ 出品者 + 貢献度バッジ |
| **主操作** | 〔出品する〕（floating primary） |

### 2.2 新規出品（listing create · 2026-06-09 レビュー反映）

![新規出品](./mockups/ihl-06-market-listing-create.png)

| チャンク | 内容 |
|----------|------|
| **観測中個体** | チェックリスト（データ自動紐づけ） |
| **その場追加** | 写真撮影 / ファイル選択（観測外の新規出品） |
| **まとめ出品** | 幼虫 12 匹等 · 複数選択可 |
| **主操作** | 〔出品する〕（緑） |

### 2.3 出品詳細 + プライベートボード（item detail + private board Stage 1）

![出品詳細 + プライベートボード](./mockups/ihl-06-market-detail-board.png)

| チャンク | 内容 |
|----------|------|
| **個体** | 大画像 / 価格 / spec / 出品者バッジ / 〔この個体に申し込む〕（主操作） |
| **プライベートボード** | **当事者2人のみ・第三者非公開**（鍵アイコン）。掲示板二人部屋（公開観覧）とは **別物**（§11.0 Stage 1） |
| **ステッパ** | マッチング → 振込 → 配送 → 評価（4 段階 · §11.0.1） |
| **支払期限** | 「残り 11日」（マッチング後 **2 週間** · Y01/Y02） |

> **評価・8% 振込案内は Stage 3（取引成立）後**。成約・`settled` 単独では発生しない（§11.0.1 · [`23-GMO-振込案内.md`](./23-GMO-振込案内.md)）。

## 3. 争い導線（評価・指摘）

- 出品への **指摘**（Y09）/ Engagement 争いは **掲示板と同型の指摘**（[`11-争い-二人部屋.md`](./11-争い-二人部屋.md) · §6.5）。
- 配送完了後（Y03/Y04/Y06）は任意で **公開エクスチェンジ → プラチナ投票**。
- 評価は **タグ + 理由必須**・版管理（新時刻が勝ち · FR-DSP-17）。

---

## 4. 状態（必須）

| 状態 | 表示 |
|------|------|
| loading | カード/ボードはスケルトン |
| empty | 「出品がありません」+ 〔出品する〕誘導 |
| error | 「読み込めませんでした」+ 再試行 |
| 未ログイン | 申込・ボードはログイン誘導 |

---

## 5. デザイン規範

- 状態チップは **意味色のみ**（出品中/オークション/成約）。
- 価格・金額は装飾しすぎない（regular weight）。
- **PII（住所・口座）はボードに直書きしない** — 設定で事前登録（FR-DSP-19 · [`../01-要件/12-設定.md`](../01-要件/12-設定.md)）。
