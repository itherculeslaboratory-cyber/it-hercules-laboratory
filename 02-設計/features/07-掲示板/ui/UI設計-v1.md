## 3. 要点

- カテゴリ: 愚痴 / 改善提案 / 論文 / 裁判 / 説明（FR-BBS-06）。
- 投稿の `…` → **引用 / 指摘**（**通報ボタンなし** · 11 §3.2）。

- 追記文化（INSERT ONLY · NFR-BBS-01）。登録言語表示・二人部屋クライアント翻訳（FR-DSP-11）。
- 状態: loading / empty / error / **投稿失敗 rescue**（raw 非表示 · FR-BBS-07）必須。
