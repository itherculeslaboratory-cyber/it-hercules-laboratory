## 1. 主タスク（1 つ）

**カルマ値とカルマカウントの現状を把握し、必要なら免罪符導線へ進む**。

---

## 2. イメージ画面

![カルマ概要](./mockups/ihl-08-karma-summary.png)

| チャンク | 内容 |
|----------|------|
| **カルマ値** | -100〜+100 レンジバー · 大きな現在値（緑/赤は意味のみ） |
| **カルマカウント** | 整数 · 黄注意で Fib ペナルティ説明リンク |
| **月次ルール** | +10/月（count=0 の月のみ）· 次回 25 日 |
| **最近の変動** | 日付 · イベント · Δ値（最大 5 件） |
| **導線** | リンク「免罪符ショップへ」→ `/economy/shop` |

**プロフィール 3 指標**（入口）: [`ihl-profile-three-metrics.png`](./mockups/ihl-profile-three-metrics.png) — カルマは **独立カード 1 枚**（ADR-H-08）。

---

## 3. 状態（必須）

| 状態 | 表示 |
|------|------|
| loading | スケルトン |
| empty（新規） | 0 / 0 + ルール説明 |
| error | 再試行 |
| 警告（&lt;-50） | 赤注意「BAN まであと N」 |

---

## 4. デザイン規範

- 二層モデル（値 vs カウント）を **別チャンク**で混同しない
- PT ショップ・貢献度カードと **同画面に統合しない**
