## 2. クリックパス

### 2.1 プロフィールから詳細

```text
/me/profile (PR) ──カルマカード──▶ /karma (08)
                                      │
                         免罪符リンク ──┼──▶ /economy/shop (22)
                                      │
                         戻る ────────┴──▶ PR
```

| Step | 操作 | クリック数（ホーム起点） |
|:----:|------|:------------------------:|
| 1 | ホーム → プロフィール（将来ナビ or 設定） | 1〜2 |
| 2 | カルマカードタップ | +1 |
| 3 | 免罪符ショップ | +1 |

### 2.2 争い・裁判からの間接遷移

```text
/board/.../dispute (11) ──（合意/判決後）──▶ 通知 ──▶ /karma (08)
```

- カルマカウント増加時は **Fib ペナルティ** を 08 画面の「最近の変動」に表示（§2.3）

### 2.3 永久 BAN

```text
カルマ値 -100 ──▶ ログアウト強制 · /login で「アカウント停止」
他ユーザー ──▶ /me/profile/:userId で「永久 BAN」バッジ（PR 状態）
```

## 3. 状態（必須）

| 状態 | 表示 | 導線 |
|------|------|------|
| loading | 値・グラフスケルトン | — |
| empty（新規） | 値 0 · count 0 · 月次ルール説明 | — |
| error | 再試行 | PR へ |
