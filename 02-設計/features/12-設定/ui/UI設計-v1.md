## 4. チャンク・状態・規範

- **3 チャンク**（局留め / 配送先 / 口座）+ 上部に理由 1 行。主要導線 **3 クリック以内**（設定 → 入力 → 保存）。
- **状態必須**: loading / 空状態（未設定: 各セクション「○ 未設定」+ 価値説明）/ 保存中（ボタン disabled）/ 保存成功（1 行）/ エラー（理由付き・raw 非表示）。
- **口座番号はマスク表示**（末尾 4 桁）。再入力時のみ全桁編集可。
- 色は意味のみ（設定済=success / 注意=warning / 情報=info）。装飾的多色化しない。
- マーケット申込からの誘導（FR-SET-19）はこの画面へ **1 クリック**で到達し、戻れる。

## 5. アクセシビリティ / 人間工学

- キーボードのみで全項目入力・保存可能（`data-testid: settings-trade-pii-save` 等）。
- 大きなタップ標的（モバイル）。ラベルと入力の関連付け（aria）。
