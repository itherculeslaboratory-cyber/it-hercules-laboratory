# 14 貢献度 — UI 設計（イメージ画面）

> **ステータス**: **草案 · 人間目視レビュー待ち**
> **正本**: [`../01-要件/14-貢献度.md`](../01-要件/14-貢献度.md) · `civilization/PlatinumCoinRules.md` §5

---

## 1. 主タスク（1 つ）

**「自分の貢献度とプラチナを把握する」**。ホーム要約バッジからの遷移先（ホーム §4 参照）。

---

## 2. イメージ画面

![貢献度 マイ貢献](./mockups/ihl-14-contribution-badge.png)

| チャンク | 内容 |
|----------|------|
| **要約バッジ** | 貢献度（1,240・上昇は緑）/ プラチナ 12枚 / 次のコインまで（progress · 100 単位 = 1 枚 · FR-CONTRIB-05） |
| **内訳** | いいね獲得 / コピー・組み込み / 上流分配 +12%（FR-CONTRIB-03） |
| **最近の加算** | イベント種別 + `+score`（緑）+ 時刻のリスト |
| **注記** | 「貢献度は減りません」（非負 invariant · スコープ外: 減算） |

---

## 3. 状態（必須）

| 状態 | 表示 |
|------|------|
| loading | バッジ/リストはスケルトン |
| empty（0） | 「まだ貢献がありません」+ いいね・投稿の導線 |
| error | 「読み込めませんでした」+ 再試行 |

---

## 4. デザイン規範

- score のみ表示・換算式の詳細は **短い説明 + ヘルプ**（NFR-CONTRIB-04 · 3 行以内）。
- 上昇トレンドのみ緑（意味色）。
- 一般ユーザーに **グラフ手動編集 UI を出さない**（運用/Kernel 側）。
- ホーム要約（[`01-ナビ・ホーム.md`](./01-ナビ・ホーム.md)）と数値整合。
