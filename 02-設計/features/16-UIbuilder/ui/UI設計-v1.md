# 16 UIbuilder — UI 設計 v1（配置・デザイン中心の IDE ワイヤー）

> **ステータス**: 草案 v1（人間レビュー待ち — 設計ゲート「UI 設計」）
> **作成日**: 2026-06-07
> **前提**: [`16-UIbuilder-詳細設計-v1.md`](./16-UIbuilder-詳細設計-v1.md) · [`16-UIbuilder-遷移設計-v1.md`](./16-UIbuilder-遷移設計-v1.md) · `ui-reference/preferences.md` §A
> **原則（ui-design-review）**: 主タスク 1 つ・3 クリック以内・1 画面 1 主ボタン・状態（loading/empty/error）必須・色は意味のみ。

---

## 1. 主タスク（この UI で完了する 1 つのこと）

**「既存部品を選んで画面に配置し、見た目を整えて保存する」**。機能を作るのではなく **組み立てる**（C10 / REFRAME）。

---

## 2. チャンク構成（3〜5 塊）

| # | チャンク | 役割 |
|---|----------|------|
| 1 | **パレット（左）** | 配置できるブロック種別 + catalog 部品（kind/layer/status フィルタ） |
| 2 | **キャンバス（中央）** | ScreenDef 配置（D&D・並べ替え）・選択ブロックの強調 |
| 3 | **プロパティ（右）** | 選択ブロックの **配置（layout）** と **デザイン（design）** と **紐づけ（binding）** |
| 4 | **プレビュー（切替）** | ライブプレビュー（実データ read-only） |
| 5 | **保存バー（上/下）** | lint→保存（INSERT ONLY）・（L4）fork Apply |
| 6 | **テーマ（切替タブ）** | ThemePack 選択 · 5 primitive プレビュー（ADR-H-17）· Phase 1 はプリセットのみ |

> RAG / Pilot / route-matrix タブは **去** — IDE 内必須にしない（FR-16-REFRAME-08）。表示が要る場合は **read-only リンク**（platform docs）。fork タブは **L4 の Apply ボタン**に縮小（プロトコル UI なし）。**テーマタブ**は FR-16-REFRAME-01 のデザイン層として **残す**（[`16-UIbuilder-テーマ.md`](../../02-設計/features/16-UIbuilder/ui/テーマ.md)）。


### 3.1 主要操作（大ボタン）

| ボタン | 役割 | 配置 |
|--------|------|------|
| **〔保存〕** | lint → INSERT ONLY 保存（**唯一の主ボタン**・primary 色） | 右上（常時） |
| 〔＋ ブロック追加〕 | キャンバスへブロック追加 | キャンバス下 |
| 〔部品を選ぶ〕 | catalog pick フロー起動 | プロパティ「紐づけ」 |
| 〔プレビュー〕 | 編集 ⇄ プレビュー切替 | ヘッダ・トグル |
| 〔公開（Apply）〕 | L4 のみ・既存テンプレ派生 | 保存後に出現（副次・通常色） |

## 4. catalog pick（紐づけ）モーダル

```text
┌─ 機能を紐づける ───────────────────────────┐
│ フィルタ: 種別[全▾] 層[L1▾] 状態[stable▾]   │
│ ┌──────────────────────────────────────┐ │
│ │ value_check_block   stable  L1  〔選択〕│ │
│ │ economy_hub_core    stable  L3  (gray) │ │  ← 層不足はグレー+理由
│ │ value_check_evaluate(connector) L2 〔選択〕│
│ └──────────────────────────────────────┘ │
│ 空状態: 利用できる部品がありません           │
│   → 新しい機能は開発パイプラインで追加（ヘルプ）│  ← 代替導線
│                          〔キャンセル〕      │
└──────────────────────────────────────────┘
```

> 3 クリック: ブロック選択 → 〔部品を選ぶ〕 → 〔選択〕（NFR-16-04）。

## 5. 状態（必須）

| 状態 | 表示 |
|------|------|
| **loading** | lint 中スピナ・保存中はボタン無効 + スピナ |
| **empty** | パレット部品ゼロ → 「部品がありません・repo で追加（ヘルプ）」/ キャンバス空 → 「ブロックを追加して始めましょう」 |
| **error** | lint 失敗バナー（平易文言）+ 該当ブロック強調。raw JSON/stack は **L1 に出さない**（NFR-16-03） |
| **saved** | 保存トースト（新 screen_def_id・版を明示） |
| **権限なし** | 層不足アクションはグレー + ツールチップ「L3 以上で利用可」 |

## 6. デザイン規範（preferences.md 整合）

| 項目 | 適用 |
|------|------|
| チャンク | 主要 3〜5（パレット/キャンバス/プロパティ + プレビュー/保存） |
| 色 | 意味のみ。primary は **保存 1 つ**。エラー=赤・情報=青・無効=グレー |
| 余白 | 広め（パネル間 32px 目安）。キャンバスは呼吸を持たせる |
| 文言 | L1 向け平易（「機能を紐づける」等）・1 行 1 情報 |
| アクセシビリティ | キーボードでブロック選択・並べ替え・保存（Tab/矢印/Enter） |

## 7. テーマタブ（ADR-H-17 · デザイン層）

配置タブと **ヘッダで排他切替**（同時編集しない）。

| 要素 | 仕様 |
|------|------|
| プリセット | `ihl-lineage-dark`（既定）· `ihl-core-light`（Phase 1 同梱） |
| プレビュー | Button / Input / Card / Tab / Badge — `ui_primitive_catalog.yaml` |
| 編集 | Phase 2: トークン行（L3+）· L1 はプリセット名のみ |
| 適用範囲 | world_default（既定 UI）· override は L4 + 確認ダイアログ |
| 境界文言 | OBS-TPL・配置は別タブ（事実ベース · `no-user-facing-unimplemented` 整合） |

詳細ワイヤー: [`../../02-設計/features/16-UIbuilder/ui/テーマ.md`](../../02-設計/features/16-UIbuilder/ui/テーマ.md)。

## 8. RAG / fork / Pilot タブの去留（FR-16-REFRAME 判断）

| 旧タブ | 去留 | 理由 |
|--------|------|------|
| RAG 検索 | **去**（IDE 必須から外す） | 機能であり配置・デザインでない |
| fork（プロトコル） | **去** → L4 の Apply ボタンのみ | プロトコルは backend 正本 |
| Pilot-B13 / route-matrix | **去** → read-only リンク | platform / CI（FR-16-REFRAME-08） |
| コード / R2 タブ | **縮小**（L3/L4 読取中心） | 生 JSON は L1 非表示 |
