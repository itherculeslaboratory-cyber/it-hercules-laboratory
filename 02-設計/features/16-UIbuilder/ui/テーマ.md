# 16 UIビルダー — テーマ編集（ThemePack）

> **ステータス**: **草案 v1 · 人間レビュー待ち**（2026-06-09）
> **目的**: UIbuilder の **デザイン層**として、世界全体の見た目（ThemePack）を選び・編集し、全画面に適用する。
> **正本 ADR**: [`../02-設計/_横断/adr/ADR-H-17-DesignTheme-テーマパック.md`](../02-設計/_横断/adr/ADR-H-17-DesignTheme-テーマパック.md)
> **関連**: [`16-UIbuilder-コンテキスト編集.md`](./16-UIbuilder-コンテキスト編集.md) · [`../02-設計/features/16-UIbuilder/ui/UI設計-v1.md`](../02-設計/features/16-UIbuilder/ui/UI設計-v1.md)
> **preferences**: `ui-reference/preferences.md` §A（色は意味のみ）· §B（血統 OS トークン）

---

## 1. 主タスク（この UI で完了する 1 つのこと）

**「世界の見た目（ThemePack）を選び、5 つの基本部品の見え方を確認して保存する」**。

配置・機能紐づけ・観測計測テンプレ（OBS-TPL）編集は **含まない**（ADR-H-17 §8）。

---

## 2. 入口

| 入口 | 権限 | 動作 |
|------|------|------|
| UIbuilder ヘッダ **〔テーマ〕** タブ | L1+ | テーマ編集 pane を開く |
| 設定 › 表示 › テーマ | 全ユーザー（Phase 2） | **プリセット選択のみ**（Phase 1） |
| グローバル文脈バー › 編集（ADR-H-14） | L3+ | Builder へ遷移し **テーマタブをフォーカス**可 |

> Phase 1: Streamlit は設定サイドバーに **プリセット 2 件**（血統ダーク / コア明るめ）のみ。

---

## 3. チャンク構成（3〜5 塊）

| # | チャンク | 役割 |
|---|----------|------|
| 1 | **プリセット一覧（左）** | 同梱・fork 済み ThemePack カード（名前 + 短い説明） |
| 2 | **トークン編集（中央）** | Phase 2: 意味色・背景・余白（カテゴリ別アコーディオン） |
| 3 | **Primitive プレビュー（右）** | Button / Input / Card / Tab / Badge のライブサンプル |
| 4 | **適用範囲（上）** | world_default / feature_override / screen_override |
| 5 | **保存バー** | lint → INSERT ONLY 保存 · （L4）fork 派生 |

---

## 5. 主要操作

| ボタン | 役割 | 層 |
|--------|------|-----|
| **〔保存〕** | ThemePack を INSERT ONLY 保存（**主ボタン**） | L1+（Phase 1 は適用のみ） |
| 〔プリセットを選ぶ〕 | 同梱 pack を world_default に設定 | L1+ |
| 〔派生を作る〕 | 親 pack から fork（新 id） | L4 · Phase 2 |
| 〔全画面プレビュー〕 | 現行 ScreenDef をテーマ適用して表示 | L2+ |

---

## 6. 状態（必須）

| 状態 | 表示 |
|------|------|
| **loading** | lint / 保存中スピナ |
| **empty** | プリセットゼロ → 「同梱テーマを読み込んでください」（開発時のみ） |
| **error** | トークン不適合 · contrast 不足 → 該当行ハイライト（raw CSS エラー非表示） |
| **contrast_warning** | WCAG AA 未満 → 黄バッジ「読みにくい組み合わせです」 |
| **saved** | 新 `theme_pack_id` トースト |

---

## 7. 境界（ユーザー向け文言の事実ベース）

| できないこと | 代替導線 |
|--------------|----------|
| 観測の計測項目テンプレを編集 | 観測 › テンプレ一覧（OBS-TPL · ADR-H-13） |
| 画面のブロック配置を変える | UIbuilder › 配置タブ |
| 新しい機能部品を追加 | 開発パイプライン（`/help/builder-capability`） |
| 写真の明るさを変える | 観測写真は色補正なし（preferences §C） |

---

## 8. デザイン規範

| 項目 | 適用 |
|------|------|
| 色 | **意味色 + 中立のみ**。プリセット外の装飾色追加 UI は出さない |
| 主ボタン | **保存 1 つ**（配置タブと同型） |
| 余白 | パネル間 32px · プレビュー周囲 40px（`--civ-space-graph`） |
| キーボード | プリセット一覧は矢印 + Enter · トークン行は Tab 移動 |
