## 2. データ契約（catalog · ThemePack · canvas）

| フィールド | 規約 | FR |

### 2.2 ThemePack（ADR-H-17 · 実装）

| フィールド | 規約 |
|------------|------|

| `tokens` | `design_token.yaml` 許可 key のみ（INV-TP-01） |
| `primitive_variant_defaults` | `ui_primitive_catalog.yaml` 検証 |

| 項目 | 契約 |

## 3. API / route 契約（IHL 実エンドポイント · retrofit）

### 3.1 `GET /api/v1/theme-packs`

| 項目 | 契約 |
|------|------|
| 200 | `{ items: ThemePack[] }` |
| 秘密値 | 返さない |

### 3.2 `POST /api/v1/theme-packs`

| 項目 | 契約 |
|------|------|
| body | ThemePack 部分または全文 |
| 200 | `{ status: saved, pack }` |
| 検証 | tokens が design_token 許可リスト内（設計 · lint gap） |

### 3.3 `GET /api/v1/theme-packs/{pack_id}`

| 項目 | 契約 |
|------|------|
| 200 | pack 全文 |
| 404 | 不明 id |

### 3.4 `POST /api/v1/builder/canvas`

| 項目 | 契約 |
|------|------|
| body | `{ canvas_id, nodes[] }` |
| 200 | `{ status: saved, manifest }` |
| 用途 | 配置・デザイン変更の保存（FR-16-REFRAME-01/05） |
