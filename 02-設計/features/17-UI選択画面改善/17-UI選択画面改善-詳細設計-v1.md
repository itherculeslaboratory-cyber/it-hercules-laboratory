## 1. コンポーネント契約

| コンポーネント | 責務 | 入出力 |
|----------------|------|--------|
| `WorldTemplatePicker` | 運営 default / recommended / custom テンプレ一覧 | IN: user_id · OUT: `template_id` |
| `TemplateLevelBadge` | default / recommended / custom 表示 | IN: level enum |
| `RequiredFeatureGate` | C-USB 必須機能欠落警告 | IN: template graph · OUT: warn[] |
| `LayerSelectorBar` | L0–L3 編集深度切替（表示のみ永続化は v2） | IN: layer · OUT: context |

## 2. データ契約（草案）

```typescript
interface WorldRoutingTemplate {
  template_id: string;
  level: "default" | "recommended" | "custom";
  author_user_id: string;
  route_graph: RouteNode[]; // C-USB 準拠
  required_features: string[]; // FR-17-02 正本リスト
  social: { likes: number; platinum_votes: number };
}
```

## 3. API（将来 · 実装禁止ゲート有効）

| メソッド | パス | 用途 |
|----------|------|------|
| GET | `/api/world-templates` | 一覧（level フィルタ） |
| POST | `/api/world-templates/:id/apply` | ユーザー設定へ適用（R2 INSERT） |
| POST | `/api/world-templates/:id/fork` | custom 複製 |

## 4. 非機能

- テンプレ切替は **3 クリック以内**（設定 → テンプレ一覧 → 適用）
- 必須機能欠落時は **ブロックではなく警告**（運営 default は常に valid）
