## 0. この文書の位置づけ

本書は **要件 #17（UI選択画面改善）の `FR-17-*` / `NFR-17-*` を「実装と検証で追える設計」に厚くした v2** である。

- **IHL スコープ**: IHL は **単一 Streamlit UI** — World テンプレ淘汰・Layer 選択文化は **civ-os 固有**。IHL 正本は **settings/preferences 投影 + 将来 world_template 契約の設計境界**。

- **legacy salvage**: `WorldRoutingManager.tsx` · `LayerSelector.tsx` · `/dev/world-routing` — dev のみ · local state。
- **v2 の追加価値**: **§2 WorldRoutingTemplate データ契約（設計）** · **§3 現行 API 契約（retrofit）** · **§7 parity gap 透明化**。
