## 1. パイプライン契約（IHL Phase 1）

```text
ingest → thumbnail(512px JPEG) → embedding(DINOv2|dummy) → manifest join → UI
```

| 段階 | 出力キー | 制約 |
|------|----------|------|
| ingest | `raw/{capture_id}.jpg` | 色補正禁止 |
| thumbnail | `thumb/{capture_id}.jpg` | 長辺 512px · 補正なし |
| embedding | `emb/{capture_id}.npy` | INSERT ONLY |
| manifest | `manifest/{session_id}.json` | 撮影条件必須 |

## 2. 種候補 UI 契約

| フィールド | 説明 |
|------------|------|
| `species_candidates[]` | confidence 降順 · **自動確定なし** |
| `vision_assist` | off / heuristic / clip / http |
| `user_confirmed_taxon` | ユーザー確定のみ Truth |

## 3. 観測入力との接続

- 計測入力 `05i` から写真アップロード → 解析結果を **種族 DD 候補**に反映（選択はユーザー）
- テンプレ一覧サムネは `thumbnail` 正本（`ihl-05-obs-template-list.png` 参照）

## 4. 非機能

- 25MB 上限 · jpeg/png/webp
- raw エラー非表示 · 再試行トースト
