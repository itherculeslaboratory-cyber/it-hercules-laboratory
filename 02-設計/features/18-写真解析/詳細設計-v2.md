## 2. データ契約（capture · embedding · tag_event）

### 2.1 capture / input manifest（ingest 後）

| フィールド | 型 | 規約 | FR |
|------------|-----|------|-----|
| `capture_id` | string | 安定 ID | FR-18-01 |
| `input_path` / R2 key | string | raw 画像参照 | FR-18-01 |
| `input_hash` | string | 再解析用 | FR-18-05 |
| `capture_conditions` | string | 撮影条件テキスト必須 | NFR-18-02 |
| `value_origin` | enum | 直接/画像/推定を混同しない | NFR-18-03 |

- **色補正禁止**（NFR-18-02）: thumbnail/raw 表示にフィルタをかけない。

### 2.2 embedding manifest + npy（`embedding_builder`）

| フィールド | 規約 | FR |
|------------|------|-----|
| `embedding_id` | `emb_{capture_id}` | FR-18-07 |
| `embedding_dim` | dummy=384 / dinov2 可変 | FR-18-07 |
| `normalized_flag` | true · L2=1 | FR-18-07 |
| `vector_offset` | manifest 内オフセット | FR-18-07 |
| R2 key | `manifests/embedding/{run_id}.json` | NFR-18-04 |

- **NaN 禁止** · **既存 key への put 禁止**（append-only run）。

### 2.3 tag_event（ユーザー承認後 · Truth）

| 操作 | 契約 |
|------|------|
| `write_tag_event` | `target_type=capture` · `action=add` · `source_type=model_inference` |
| 自動確定 | **行わない** — approve API 経由のみ Truth 化 |

### 2.4 thumbnail / QC / searchable set（v1 設計 · gap）

[`18-写真解析-詳細設計-v1.md`](./18-写真解析-詳細設計-v1.md) · legacy `2026.06-06-legacy` §8.2–8.3:

- thumbnail: 長辺 512px JPEG · EXIF transpose · 補正なし
- QC: blur/exposure/scale → `qc_flag` · `qc_score`
- searchable_capture_set: Polars join · latest pointer

## 3. API / route 契約（IHL 実エンドポイント · retrofit）

### 3.1 `GET /api/v1/photo-analysis/result`

| 項目 | 契約 |
|------|------|
| 200 | `{ status, tags[], capture_conditions, capture_id, photo_url? }` |
| tags[] | `{ tag, confidence, tag_type }` — **参考順のみ** |
| NFR | raw エラー非表示 · 撮影条件併記 |

### 3.2 `POST /api/v1/photo-analysis/approve`

| 項目 | 契約 |
|------|------|
| body | `{ capture_id, actor_id?, tags: [{ tag, tag_type, confidence }] }` |
| 200 | `{ status: approved, tag_event_ids[] }` |
| 副作用 | `tag_event` append-only（空 tag はスキップ） |

### 3.3 `POST /api/v1/pipeline/embedding`

| 項目 | 契約 |
|------|------|
| query | `capture_id` · `run_pipeline=true\|false` |
| 200 | `{ status: queued\|skipped, capture_id, component: embedding_builder }` |
| 備考 | 本番は component CLI 非同期起動 · 現行はキュー応答スモーク |
