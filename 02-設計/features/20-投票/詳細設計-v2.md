## 2. データ契約（vote_event · poll · tally）

### 2.1 vote_event（`governance/vote_event` · INSERT ONLY）

| フィールド | 型 | 規約 | FR/NFR |
|------------|-----|------|--------|
| `vote_event_id` | string | 一意 · `vote_*` | NFR-20-02 |
| `poll_id` | string | 投票対象 poll | FR-20-01 |
| `voter_id` | string | `@handle` 平文（schema §6 #8） | schema-pack |
| `choice` | enum | approve / reject 等 | FR-20-02 |
| `created_at` | ISO8601 | append 時刻 | NFR-20-02 |

- **UPDATE/DELETE 禁止**（R2 INSERT ONLY）。

### 2.2 poll catalog（読取 · デモ/seed）

| フィールド | 説明 |
|------------|------|
| `vote_id` | poll 識別子 |
| `title` · `layer` | レイヤー 0–3 のみ（FR-20-01） |
| `options[]` | `{ option_id, label }` |

### 2.3 tally 投影

- `vote_tally(poll_id, options)` — `governance/vote_event` JSONL 走査。
- **public_tally: true** — API 応答に含める（FR-20-02 透明性）。

## 4. 状態機械（投票 UI · 自然淘汰 · 遷移 v1 整合）

```
[list polls] ──GET votes──▶ [poll_ready]
[poll_ready] ──select option──▶ [casting] ──POST ballot──▶ [voted]
[voted] ──refresh tally──▶ [poll_ready]
```

- **レイヤー 4** 対象 poll — **catalog に含めない**（FR-20-01）。
- **自然淘汰**（FR-20-05）: Governance §10 指標定義 — IHL は vote_event 記録のみ · ダッシュボード **gap**。
- **免罪符フロー**（遷移 v1）: `/karma` → `/economy/shop` — **#22 正本** · 本機能 API 外。
