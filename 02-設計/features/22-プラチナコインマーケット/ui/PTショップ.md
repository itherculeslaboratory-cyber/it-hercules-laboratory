# 22 プラチナコインショップ — UI 設計（イメージ画面）

> **ステータス**: **草案 · 人間目視レビュー待ち**
> **正本**: [`../01-要件/22-プラチナコインマーケット.md`](../01-要件/22-プラチナコインマーケット.md) · 効果は [`08-カルマシステム.md`](../01-要件/08-カルマシステム.md) §4

---

## 1. 主タスク（1 つ）

**「PT を消費して免罪符を購入する」**。投票（§20）とはスコープ分離した **PT 消費型ショップ**。

---

## 2. イメージ画面

![PT ショップ](./mockups/ihl-22-pt-shop.png)

| チャンク | 内容 |
|----------|------|
| **PT 残高** | 右上ピル「残高 12 PT」（投票・指摘課金と **同一台帳** · FR-PTMKT-04） |
| **商品カード** | 黄金ヘラクレス教の免罪符 / 効果「カルマカウント -1（0 未満なし）」 |
| **価格** | **次回購入価格 Fib(n) PT**（初回 1 PT · 価格段階 n 表示 · FR-PTMKT-05/08） |
| **価格ヒント** | 購入ごとに 1→1→2→3→5→8 PT 上昇 / 毎月 1 段階下がる |
| **主操作** | 〔購入する（1 PT）〕 |
| **購入履歴** | 日時 / 価格 / 段階 のリスト |

> 上限なし — 連続購入は Fib 価格上昇で **自然に抑制**（FR-PTMKT-07 · §5.4）。

---

## 3. 状態（必須）

| 状態 | 表示 |
|------|------|
| loading | カードはスケルトン |
| empty | 購入履歴なし → 「まだ購入はありません」 |
| error | 「購入できませんでした」+ 再試行 |
| **PT 不足** | 理由明示で拒否（FR-PTMKT-01）+ PT 入手導線（GMO 入金 → [`23-GMO-振込案内.md`](./23-GMO-振込案内.md)） |

---

## 4. デザイン規範

- 単一商品にフォーカス（カタログを増やしすぎない）。
- 価格段階・次回価格を明示（指数的上昇を隠さない）。
- 効果は **count -1 のみ**（カルマ値の直接購入はしない · §10）。
- 購入は R2 INSERT ONLY。
