### 2.2 価格段階 n（設計正本 · IHL gap）

## 4. 状態機械（購入 · stub vs 設計）

```
[閲覧] shop_view → balance + catalog + 次回 Fib(n) 表示（設計 · UI gap）
[購入] PT >= price → pt_event(-price) → （設計）karma count -1 → n += 1
[月次] 未購入暦月 → n -= 1（設計 · バッチ gap）
```
