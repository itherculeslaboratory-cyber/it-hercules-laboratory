## 3. 要点

- 情報バナー → 振込先 / 金額 / **振込コード `U-XXXX`**（各行コピー · §2.2 `deriveTransferCode(userId)`）。
- 名義末尾追記の注意（例 `ﾔﾏﾀﾞ U-4F9A` · GMO 48 半角文字以内 · §2.3）。
- ユーザーごと **1 コード**（8% も PT も同一 · §2.4）。複数 pending は **入金日時 FIFO**（§2.5）。
- 状態: loading / 照合待ち / 照合済 / error（短い理由 · NFR-GMO-04）。
- 秘密値は画面・ログに出さない（NFR-GMO-01）。本番実入金は人間ゲート（FR-GMO-07）。
