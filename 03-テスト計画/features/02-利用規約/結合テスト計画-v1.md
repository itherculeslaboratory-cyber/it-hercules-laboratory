## 1. 目的・範囲

規約 route（`/api/v1/terms` · `/api/v1/legal/agree`）が **event_store · 版メタ契約と整合**して動くことを `fastapi.testclient.TestClient` で担保する。

| 範囲 | 対象 |
|------|------|
| In | 未認証閲覧 · 同意 POST · 副作用（legal/v1 JSON 追記）· 応答契約 |
| Out | 関数内部分岐（単体）· UI 二層タブ/resolver（gap）· 法務 binding（human） |

## 4. カバレッジ方針

- route × 副作用の **HTTP 契約 + ファイル追記**（FR-10/13/14 · NFR-05）を中心に担保。
- ステータスコードだけでなく **応答 body / 追記 JSON 内容**を assert（C3 準拠）。
- 二層表示 · resolver · scroll gate は v2 §7 P4 **設計ギャップ** — RTM `status=gap`/`deferred`。
