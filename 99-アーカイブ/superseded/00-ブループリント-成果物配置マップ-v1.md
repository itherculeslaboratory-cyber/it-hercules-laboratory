## 2. 世界観アセット配置（ユーザー所有）

```text
02-設計/_ui-global/assets/
  README.md                 ← 全体ルール
  brand/                    ← ロゴ・マーク
    README.md
  economy/                  ← プラチナコイン・免罪符
    README.md
  metrics/                  ← カルマ・貢献度・市場評価・空状態
    README.md
```

- **生成**: AI は行わない（一覧・仕様のみ）  
- **ライセンス**: ユーザー提供 or 将来生成時に一覧へ追記  
- **参照**: UI 設計 md は `assets/{category}/{asset-id}.{ext}` を相対パスで指す

## 4. 新規成果物の追加手順

1. 本マップに行を追加（または既存行の状態を更新）  
2. `01-要件/README.md` 索引に 1 行リンク  
3. モックを追加する場合: `archive-mock.mjs` → `walkthrough.js` → `00-画面一覧-全体像.md`  
4. 監査対象なら `00-監査役-実装前ゲート-v1.md` に証跡行を追加
