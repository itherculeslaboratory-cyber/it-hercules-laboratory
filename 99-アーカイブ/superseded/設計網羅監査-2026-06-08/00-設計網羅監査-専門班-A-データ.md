## 付録 A. Truth / Snapshot / Derived の全体マップ

```text
Truth（append-only JSONL · R2のみ）
├── Individual, Capture, Measurement, Observation
├── Cross（cross_status 変化の Event 設計 → P0-DATA-03）
├── CrossParent（parent_role = Template 定義 → P1-DATA-04）
├── OffspringAssignment（confidence の採用ロジック → P0-DATA-04）
├── AnnotationEvent, LabelEvent（source_type 付き）
├── ModelRun（監査用 Truth）
└── life_event（Template 定義 event_type）

  ↓ Snapshot 再計算

Snapshot
├── searchable_capture_set / individual_master
├── Lineage（ancestor/descendant）← 生成 pipeline 未設計
├── cross_summary / cross_growth_summary
├── ModelEvaluation
└── Knowledge Graph Snapshot ← スキーマ未定義

  ↓ 派生成果物

Derived
├── Training Dataset ← export pipeline 未設計 P0-DATA-05
├── Model Artifact
└── Embedding, Index（FAISS Phase 2）
```
